[ngClass]

is a directive that adds a classname in an element, depending on a condition: [ngClass]={odd: condition}
Output -> {{ number }} |
numbers = [1,2,3,4,5,6,7,8]

[ngStyle]

is a directive that adds a specific style in an element (+condition based): [ngStyle]={backgroundColor: string}

Output -> {{ number }} |
fruits = ['Apple', 'Orange', 'Banana']

Custom Attribute Directives

  1. Highlights the element with yellow color
  2. Adds 5px padding
  3. Makes the font-weight -> bold
  4. Border radius of 8px
